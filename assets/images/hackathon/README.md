# AWS Hackathon Photos Album

Place your AWS AI Hackathon photos here.

## Naming Convention

Simply number your photos:

- `1.jpg`, `2.jpg`, `3.jpg`, etc.

## Supported Formats

- JPG, JPEG, PNG, GIF

## Example

```
assets/images/hackathon/
├── 1.jpg
├── 2.jpg
├── 3.jpg
├── 4.jpg
└── 5.jpg
```

The photos will automatically appear in the gallery on the hackathon page!
