# Co-op Photos Album

Place your HAECO co-op photos here.

## Naming Convention

Name your files with category prefix:

- `hackathon_1.jpg`, `hackathon_2.jpg` - AWS Hackathon photos
- `leanday_1.jpg`, `leanday_2.jpg` - Lean Day photos
- `techathon_1.jpg`, `techathon_2.jpg` - Techathon photos
- `events_1.jpg`, `events_2.jpg` - Other events
- `reinvent_1.jpg`, `reinvent_2.jpg` - AWS re:Invent photos

## Supported Formats

- JPG, JPEG, PNG, GIF

## Example

```
assets/images/coop/
├── hackathon_1.jpg
├── hackathon_2.jpg
├── leanday_1.jpg
├── leanday_2.jpg
├── reinvent_1.jpg
└── events_1.jpg
```

The photos will automatically appear in the gallery on the co-op page!
